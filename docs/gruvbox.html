gruvbox.style module¶

A retro groove color scheme for Pygments.

As originally introduced as a Vim colorscheme. [1]

This pygments colorscheme can be used with any API that allows for Pygments syntax highlighting. It has; however, been primarily tested with IPython.

Extends previous repositories that have ported Gruvbox to Pygments, and adds in new definitions for different tokens.
